# ci4roles-fcm

Send Notification using Codeigniter 4 and Firebase Cloud Messaging (FCM)

### Installing
```
* composer install
* setting configuration hostname, username, password and database in path Config/Database.php or .env files
```
### Executing program
```
* php spark migrate
* php spark serve
```
